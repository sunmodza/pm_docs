# Error Handling

API error codes and handling.

## Error Response Format
```json
{
  "error": "Error message description",
  "code": "ERROR_CODE",
  "details": {}
}
```

## Common Error Codes
- `400` - Bad Request
- `401` - Unauthorized
- `404` - Not Found
- `500` - Internal Server Error
